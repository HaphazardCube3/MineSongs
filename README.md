## Controls
- **M** - Open Music Player GUI
- **P** - Play/Pause current track
- **N** - Skip/Stop current track

## Installation
1. Download the latest release
2. Place in your `mods` folder
3. Install [yt-dlp](https://github.com/yt-dlp/yt-dlp) for YouTube support
4. Optional: Install FFmpeg for better audio conversion

## Usage
1. Press **M** to open the music player
2. Paste any YouTube URL
3. Click **Play** and enjoy!

## Requirements
- Minecraft 1.21+
- Fabric Loader
- yt-dlp (for YouTube playback)
